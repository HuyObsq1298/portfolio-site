## üíª How to view this project locally
If you are a developer or engineer and would like to download the source files to run the website locally on your own machine, follow these steps:

**1. Clone the repository**
Open your terminal and run:
`git clone https://github.com/HuyObsq1298/portfolio-site.git`

**2. Open the directory**
Navigate into the downloaded folder:
`cd portfolio-site`

**3. Launch the site**
Since this is a static HTML/CSS site, no server installation or build process is required. Simply double-click the `index.html` file in your file explorer to open it in your default web browser.
